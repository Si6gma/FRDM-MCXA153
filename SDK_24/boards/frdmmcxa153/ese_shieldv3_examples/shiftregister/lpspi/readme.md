Overview
========
Assignment for controlling a HC595 shift register with the LPSPI module.

Details
====================
Do you want to know more details about this example? Check the [details](./readme_details.md){:target="_blank"}.

Hardware requirements
=====================
- FRDM-MCXA153 board
- Type-C USB cable
- Shield V3

Board settings
==============
- Default

Preparation
===========
- Connect the type-C USB cable between the host PC and the MCU-Link port (J15) on the target board.
- Build the application.
- Run or debug the application.

Result
======
- LED1 to LED8 are on/off as controlled by the application.

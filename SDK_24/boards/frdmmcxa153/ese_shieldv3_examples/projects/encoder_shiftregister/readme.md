Overview
========
Use the rotary encoder and the shift register.

Hardware requirements
=====================
- FRDM-MCXA153 board
- Type-C USB cable
- Shield V3

Board settings
==============
- Default

Preparation
===========
- Connect the type-C USB cable between the host PC and the MCU-Link port (J15) on the target board.
- Build the application.
- Run or debug the application.

Result
======
- At startup LED1 is on. The LEDs move with the turns of the rotary encoder.

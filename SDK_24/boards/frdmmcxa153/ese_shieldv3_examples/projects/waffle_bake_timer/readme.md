Overview
========
A waffle baking timer. It is implemented using a finite statemachine and uses several of the hardware components of the Shield V3.

Hardware requirements
=====================
- FRDM-MCXA153 board
- Type-C USB cable
- Shield V3

Board settings
==============
- Default

Preparation
===========
- Connect the type-C USB cable between the host PC and the MCU-Link port (J15) on the target board.
- Build the application.
- Run or debug the application.

Result
======
- Open the VCOM (115200-8n1) in a terminal application of your choice.
- The application shows logging information of the statemachine.
